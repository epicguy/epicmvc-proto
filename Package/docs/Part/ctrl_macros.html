MACROS

MACROS are simply shortcuts or aliases that are defined by the 'Controller' to simplify programming. They have the exact same syntax and follow the same rules as 'CLICKS'. Macros are especially useful if you have to perform the same logic in multiple places throughout the application. They are defined once at the top of the 'Controller' and can be referenced as a macro anywhere else.

Example

window.EpicMvc.app$docs=
	OPTIONS:
		...
	MODELS:
		...
	MACROS:
		start: call: 'User/check', RESULTS: [
			{ r:{ valid: 'no'},	go: 'logged_out//' }
			{ r:{ valid: 'yes'}, 	go: 'logged_in//' }
		]
		go_billing_select: call: 'Billing/select_plan_default', go: 'myprofile/bill/select'
		close_modal: call:'Pageflow/restore_path'
	FLOWS:
		...
		TRACKS:
			...
			STEPS:
				...
				privacy: page: 'privacy_policy', CLICKS:
					close_modal:	macro: 'close_modal'
					cancel:		macro: 'close_modal'
				terms: page: 'terms_of_use', CLICKS:
					close_modal:	macro: 'close_modal'
					cancel:		macro: 'close_modal'
				...

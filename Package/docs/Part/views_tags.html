Other Tags

The following additional tags are available:

<epic:defer> 
	# put your javascript in here. You can include EpicMvc tags (such as <epic:foreach>) and variable references.
	$('body').scrollspy({ target: '.nav' });
</epic:defer>

<epic:comment> 
	# Comments normally are stripped before sending to the customer. Use this tag to send the text between the tags. 
</epic:comment>

<epic:define name="cool">Wow,  you just created a new epic tag called 'cool'. Use it anywhere you use other tags.</epic:define>

<epic:tag name="cool"/>
